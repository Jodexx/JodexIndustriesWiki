**JGuiWrapper** використовує [Adventure](https://docs.advntr.dev/) бібліотеку для серіалізації строк у компоненти Minecraft.

## Типи серіалайзерів
Для зручної серіалізації, було створено 
[SerializerType](https://github.com/Jodexx/JGuiWrapper/tree/master/api/src/main/java/com/jodexindustries/jguiwrapper/api/text/SerializerType.java) enum,
що містить в собі усі типи обробників з [Adventure](https://docs.advntr.dev/), включаючи [MiniMessage](https://docs.advntr.dev/minimessage/index.html)

:::warning
**JGuiWrapper** не містить в собі ніяких серіалайзерів, а лише предоставляє до них зручний доступ. Деякі з них вже включені у ядро сервера, усі інші потрібно додавати вручну у свій проект.

У разі відсутності серіалайзера на сервері, результат серіалізації/десеріалізації буде пустим.
:::

### Приклад використання
```java
Component component = SerializerType.LEGACY_AMPERSAND.deserialize("&6Some &9legacy &dtext");
```

## Стандартний серіалайзер

**JGuiWrapper** за замовчуванням використовує 
[LegacyComponentSerializer](https://jd.advntr.dev/text-serializer-legacy/4.24.0/net/kyori/adventure/text/serializer/legacy/LegacyComponentSerializer.html), 
тобто [SerializerType#LEGACY_AMPERSAND](https://github.com/Jodexx/JGuiWrapper/tree/master/api/src/main/java/com/jodexindustries/jguiwrapper/api/text/SerializerType.java#L32).

Серіалайзер за замовчуванням використовується у заголовках меню та [ItemWrapper](https://github.com/Jodexx/JGuiWrapper/tree/master/api/src/main/java/com/jodexindustries/jguiwrapper/api/item/ItemWrapper.java)

Для того, щоб змінити серіалайзер за замовчуванням, можна використати метод [GuiApi#defaultSerializer(SerializerType)](https://github.com/Jodexx/JGuiWrapper/tree/master/api/src/main/java/com/jodexindustries/jguiwrapper/api/GuiApi.java#L145)

```java
api.defaultSerializer(SerializerType.MINI_MESSAGE);
```
